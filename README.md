# telebot_py
telegram bot with telebot in python
( pyTelegramBotAPI )


#env and run
setup venv :
````
python -m venv venv
````

run venv :
````
venv\scripts\activate
````

install packages :
````
pip install -r requirements.txt
````

for run use this:
````
python core\main.py
````

copy and make Env file:
````
get telegram bot api token and replace
simple.env  to .env
````
